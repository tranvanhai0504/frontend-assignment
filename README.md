## Get started

***Setup Project***
- Clone this repo:
  ```bash
   git clone https://github.com/tranvanhai0504/frontend-assignment.git
  ```
- Move to project
  ```bash
  cd blockchain-intergration
  ```
- Install dependencies (use --fore to overwrite the conflict code on Sui sdk):
  ```bash
   npm i --force
  ```
- Run project on your local:
   ```bash
  npm run dev
  ```
